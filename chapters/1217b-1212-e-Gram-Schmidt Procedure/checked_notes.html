11.1 一、课程回顾：向量空间与信号空间的统一视角

上节课的核心目标是建立一个几何化的信号处理框架：将抽象的“信号”视为某种“向量”，从而借助线性代数的强大工具进行分析与设计。

11.1.1 1.1 向量空间的核心概念

  • 内积 (Inner Product)：衡量两个向量之间的“相似度”。对于向量 \(\mathbf{u}, \mathbf{v}\)，其定义为 \(\langle \mathbf{u}, \mathbf{v} \rangle\)。
  • 范数 (Norm)：表示向量的“长度”，计算公式为 \(\|\mathbf{v}\| = \sqrt{\langle \mathbf{v}, \mathbf{v} \rangle}\)。
  • 正交性 (Orthogonality)：若 \(\langle \mathbf{u}, \mathbf{v} \rangle = 0\)，则两向量正交。
  • 正交归一基 (Orthonormal Basis)：一组相互正交且每个向量范数为1的基底 \(\{\phi_1, \dots, \phi_n\}\)。

11.1.2 1.2 Gram-Schmidt 正交化过程

给定任意一组线性无关的向量或信号，Gram-Schmidt 算法可系统地构造出一组正交归一基。

💡 思路与方法
该过程的核心思想是“投影减法”： 1. 取第一个向量并归一化，得到 \(\phi_1\)； 2. 对于第二个向量 \(s_2(t)\)，计算其在 \(\phi_1\) 上的投影：\(\text{proj}_{\phi_1}(s_2) = \langle s_2, \phi_1 \rangle \phi_1\)； 3. 将 \(s_2\) 减去该投影，获得与 \(\phi_1\) 正交的新向量：\(v_2 = s_2 - \text{proj}_{\phi_1}(s_2)\)； 4. 归一化 \(v_2\) 得到 \(\phi_2\)； 5. 以此类推，第 \(k\) 个基向量为： \[ v_k = s_k - \sum_{j=1}^{k-1} \langle s_k, \phi_j \rangle \phi_j, \quad \phi_k = \frac{v_k}{\|v_k\|} \]

⚠️ 关键洞察
最终生成的正交基数量等于原始信号组所张成空间的维度。即使给出三个信号，若它们线性相关，Gram-Schmidt 过程将产生少于三个有效基向量，说明信号空间维度小于3。

11.2 二、信号空间：将波形转化为向量

我们将向量空间的概念推广至连续时间信号领域，构建信号处理的几何语言体系。

11.2.1 2.1 信号空间的基本定义

  • 内积：对两个能量信号 \(x_1(t), x_2(t)\) 定义为： \[ \langle x_1(t), x_2(t) \rangle = \int_{-\infty}^{\infty} x_1(t) x_2^*(t) dt \] （注：“contre”应为共轭 conjugate，“do”应为“inner product”）
  • 能量 (Energy)：信号的能量即为其范数平方：\(\mathcal{E}_x = \|x(t)\|^2 = \langle x(t), x(t) \rangle\)。
  • 正交信号：当 \(\langle x_1(t), x_2(t) \rangle = 0\) 时称二者正交。
  • 正交归一信号集：信号之间两两正交，且每个信号能量为1。

11.2.2 2.2 信号的正交展开 (Orthonormal Expansion)

这是连接连续信号与离散向量的关键桥梁。

📚 核心公式
给定 \(\mathcal{L}_2\) 空间的一组正交归一基 \(\{\phi_1(t), \phi_2(t), \cdots\}\)，任何信号 \(x(t)\) 都可展开为： \[ x(t) = \sum_{j} x_j \phi_j(t) \] 其中系数由内积确定： \[ x_j = \langle x(t), \phi_j(t) \rangle = \int x(t) \phi_j^*(t) dt \] 该等式在均方误差趋于零的意义下成立。

🎯 深入理解
信号 \(x(t)\) 与其向量表示 \(\mathbf{x} = [x_1, x_2, \dots]\) 在能量层面完全等价。因此，我们可以在信号域与向量域之间自由切换，实现问题的简化与直观化。

11.2.3 2.3 经典实例：采样定理的几何解读

采样本质上是一种基于特定基函数的正交展开。

  • 时限信号 (Time-Limited)：持续时间为 \([-T/2, T/2]\) 的信号，可用如下函数作为正交归一基： \[ \phi_k(t) = \frac{1}{\sqrt{T}} e^{j 2\pi k t / T} \cdot \text{rect}(t/T) \] 这正是傅里叶级数的基函数。展开系数 \(\alpha_k = \langle v(t), \phi_k(t) \rangle\) 即为傅里叶系数。

  • 带限信号 (Band-Limited)：频带限制在 \(W\) 内的信号，其正交归一基为平移的 sinc 函数： \[ \phi_n(t) = \text{sinc}\left(\frac{t}{T} - n\right), \quad T = \frac{1}{2W} \] 展开系数 \(x_n = x(nT)\) 即为采样值。重构公式： \[ x(t) = \sum_n x(nT) \phi_n(t) \] 完美体现了奈奎斯特采样定理。

✨ 教授洞见
“你看，采样不是魔法，它是信号空间理论的一个自然结果。我们只是选择了一组特殊的基（sinc函数），然后用内积（积分）去提取坐标（采样点）。”

11.3 三、信道编码：数字通信的“逆源编码”

11.3.1 3.1 信道编解码的整体流程

完整的数字通信链路如下所示：

信息源 -> [信源编码] -> Bit Sequence -> [信道编码] -> 基带波形 -> [带通调制] -> 信道 -> ...
... -> [带通解调] -> 噪声基带波形 -> [信道解码] -> Bit Sequence -> [信源解码] -> 信息

🔄 核心关系
信道编码与信源编码是互逆的操作： - 信源编码：将连续波形 \(s(t)\) 转换为离散比特流 \(\{b_k\}\) —— 实现数字化压缩； - 信道编码：将离散比特流 \(\{b_k\}\) 映射为适合传输的连续波形 \(s(t)\) —— 实现物理层承载。

❗ 教授强调
“它们的目的完全不同！信源编码追求高效压缩，而信道编码旨在保障可靠传输。本节重点在于如何从比特流生成可传输的波形，即‘调制’部分。”

11.4 四、数字调制：从比特到波形的艺术

11.4.1 4.1 调制基本框图

调制器主要完成两个步骤： 1. 符号映射 (Symbol Mapping)：将输入比特序列 \(\{b_k\}\) 映射为符号序列 \(\{u_k\}\)； 2. 基带调制 (Baseband Modulation)：将符号序列转换为实际传输的基带波形 \(u(t)\)。

数学表达式为： \[ u(t) = \sum_{k} u_k p(t - kT_s) \] 其中： - \(u_k\)：第 \(k\) 个时刻的符号（实数或复数）； - \(p(t)\)：基础脉冲波形 (Basic Pulse Waveform)，如矩形波、sinc函数； - \(T_s\)：符号间隔 (Symbol Duration)。

🔍 教授点拨
“这个求和过程，实质上是用符号 \(u_k\) 对平移后的脉冲 \(p(t-kT_s)\) 进行幅度加权。因此，这类调制统称为脉冲幅度调制 (PAM)。”

11.4.2 4.2 关键参数定义

  • 比特率 (Bit Rate, \(R_b\))：每秒传输的比特数，单位 bps。定义为 \(R_b = 1/T_b\)，其中 \(T_b\) 为比特间隔。
  • 符号率 (Symbol Rate, \(R_s\))：每秒传输的符号数，单位 Baud 或 symbols/s。\(R_s = 1/T_s\)。
  • 比特与符号的关系：若每 \(b\) 个比特映射为一个符号，则 \(M = 2^b\)，\(T_s = b \cdot T_b\)，故有 \(R_s = R_b / b = R_b / \log_2 M\)。

📝 示例
若比特率 \(R_b = 1\) Mbps，采用 4-PAM 调制（\(M=4, b=2\)），则符号率为 \(R_s = 0.5\) MBaud，符号间隔 \(T_s = 2\) μs。

11.5 五、星座图 (Constellation Diagram)：调制设计的画布

11.5.1 5.1 什么是星座图？

星座图是一个二维平面图，用于直观展示所有可能的符号 \(u_k\) 及其对应的比特组合。

🌌 趣闻解释
“为什么叫‘星座图’？因为它就像夜空中的星座，由一个个代表星星的点（符号）构成。比如猎户座有腰带三星，我们的 4-PAM 星座图也有四个点排成一线。”

11.5.2 5.2 M-PAM 星座图设计

  • 标准结构：\(M\) 个星座点对称分布在实轴原点两侧，相邻点间距为 \(d\)。
    • 例如，4-PAM：\(u_k \in \{-3d/2, -d/2, d/2, 3d/2\}\)。
  • 映射策略：需明确定义比特组合到星座点的规则。使用格雷码 (Gray Coding) 可使相邻点仅有一位比特不同，从而降低误比特率。

⚙️ 设计考量
M-PAM 设计需综合考虑以下关键性能指标：

指标 计算方式 目标
能量效率 (Energy Efficiency) \(R_b / E_b\) 最大化
带宽效率 (Bandwidth Efficiency) \(R_b / B\) 最大化
可靠性 (Reliability) 误检测概率 \(P_e\) 最小化

11.6 六、M-PAM 的平均能量分析

11.6.1 6.1 平均符号能量 (\(E_s\)) 计算

符号 \(u_k\) 是一个随机变量，其统计特性取决于输入比特的概率分布及映射方式。

  • 一般情况： \[ E_s = \mathbb{E}[|u_k|^2] = \sum_{m=1}^{M} P_m \cdot a_m^2 \cdot E_p \] 其中：

    • \(P_m\)：第 \(m\) 个符号被发送的概率；
    • \(a_m\)：第 \(m\) 个星座点的幅度；
    • \(E_p = \int |p(t)|^2 dt\)：基础脉冲 \(p(t)\) 的能量。

  • 特殊情况（比特等概独立）：此时各符号等概率出现，\(P_m = 1/M\)，因此： \[ E_s = \frac{E_p}{M} \sum_{m=1}^{M} a_m^2 \]

    对于标准对称 M-PAM 星座图，经推导得： \[ E_s = \frac{d^2 (M^2 - 1)}{12} E_p \]

📊 深入理解
- \(M\) 的影响：随着 \(M\) 增大，星座图跨度变宽，平均能量升高。高阶调制以更高的能量消耗换取更高的数据速率。 - \(d\) 的影响：增大 \(d\) 提升抗噪能力，但也增加发射能量需求。

11.6.2 6.2 能量与可靠性的根本权衡

⚖️ 核心矛盾
- 从能量效率角度：希望 \(d\) 小，以降低 \(E_s\)，提高能效； - 从可靠性角度：希望 \(d\) 大，以便接收机更易区分相近符号，降低误检概率 \(P_e\)。

结论：在固定发射功率（即固定 \(E_s\)）条件下，增大 \(M\) 必然导致 \(d\) 减小，从而牺牲可靠性来换取更高的带宽效率。这是数字通信系统设计中最根本的权衡之一。

🎯 教授总结
“通信系统的艺术，就在于在效率（速率、带宽、能量）和可靠性（误码率）之间找到最佳平衡点。没有完美的方案，只有最适合应用场景的设计。”
