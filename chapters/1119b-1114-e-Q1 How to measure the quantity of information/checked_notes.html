3.1 一、课程引入：从“惊讶程度”到信息量的量化

3.1.1 1. 核心问题：如何衡量一个事件的信息量？

我们常说“这个消息真惊人！”或“这结果我早猜到了”，背后其实隐藏着一个根本性问题：信息是否可以被量化测量？

🎯 教授点拨：
“信息量并非由事件本身决定，而是由它的‘不确定性’决定。”
—— 这是信息论的第一性原理。

  • 若某事件几乎必然发生（如“太阳明天升起”），其带来的信息量趋近于零。
  • 若某事件极不可能却真实发生（如“中国队赢了巴西队”），它带来巨大冲击，信息量极高。

因此，信息量 ∝ 不确定性 ∝ 惊讶程度。

3.2 二、直观理解：用“提问次数”衡量不确定性

3.2.1 1. 思想实验：Alice 与 Bob 的通信游戏

设想如下场景： - Alice 是发送端，执行一项随机实验（如抛硬币、选字母）。 - Bob 是接收端，对结果一无所知。 - Alice 只能回答 0 或 1（1=Yes, 0=No）。 - Bob 的目标是以最少的问题数准确推断出实验结果。

✅ 核心洞察：
平均所需最少提问次数，即为该事件所含信息量的度量。
因每个问题传递1比特信息，此数值也等于传输所需的最小平均比特数。

3.2.1.1 示例1：单次公平抛硬币

  • 结果：Head 或 Tail，等概率。
  • Bob 只需问一个问题：“是 Head 吗？”
  • 答案为 1 或 0 即可确定结果。
  • ✅ 信息量 = 1 比特

3.2.1.2 示例2：连续抛10次独立硬币

  • 每次独立，各有1比特信息。
  • 总信息量 = 10 × 1 = 10 比特
  • 需提出10个问题（例如，“第i次是Head吗？”）

3.2.1.3 示例3：从26个英文字母中随机选取一个

  • 最优策略：采用二分法（Binary Search）
    • 问：“在A-M之间吗？” → 缩小一半
    • 再问：“在A-F之间吗？” → 继续缩小
    • ……
  • 平均最少问题数 = \(\log_2{26} \approx 4.7\)
  • ✅ 每个字母的信息量 ≈ 4.7 比特

💡 教授点拨：
“你有没有发现？信息量和样本空间的大小密切相关。空间越大，越难预测，不确定性越高。”
因此，信息的基本单位是 比特（bit），源于二进制决策过程。

3.3 三、信息熵（Entropy）：不确定性的数学定义

3.3.1 1. 离散随机变量的信息熵 \(H(X)\)

设离散随机变量 \(X\)，取值空间为 \(\mathcal{X}\)，大小为 \(M\)，概率质量函数为 \(p(x)\)。

则其信息熵定义为：

\[ H(X) = \sum_{x \in \mathcal{X}} p(x) \log_2 \frac{1}{p(x)} = -\sum_{x \in \mathcal{X}} p(x) \log_2 p(x) \]

也可写作期望形式： \[ H(X) = \mathbb{E} \left[ \log_2 \frac{1}{p(X)} \right] \]

🔹 物理意义：
\(H(X)\) 表示完全描述随机变量 \(X\) 所需的平均最小比特数，即其内在的不确定性总量。

3.3.2 2. 信息熵的性质与推导

3.3.2.1 (1) 单位

  • 若以 2 为底，单位是 比特（bit）
  • 若以自然对数 e 为底，单位是 奈特（nat）
  • 换算关系：\(1 \text{ nat} = \log_2 e \approx 1.44 \text{ bits}\)

✅ 教授强调：
“除非特别说明，我们默认使用 bit 作为单位。”

3.3.2.2 (2) 非负性

  • 因为 \(0 \leq p(x) \leq 1\)，所以 \(\log_2 \frac{1}{p(x)} \geq 0\)
  • 故 \(H(X) \geq 0\)

3.3.2.3 (3) 确定性事件的熵为0

  • 若某个 \(x\) 满足 \(p(x) = 1\)，其余为0，则 \(H(X) = 0\)
  • ✅ 符合直觉：无不确定性，无需任何信息即可描述。

3.3.2.4 (4) 最大熵原理：等概分布时熵最大

对于固定大小 \(M\) 的样本空间，当所有取值等概率时，熵达到最大：

\[ H(X) \leq \log_2 M \]

且等号成立当且仅当 \(p(x) = \frac{1}{M}, \forall x \in \mathcal{X}\)

🔍 证明思路（利用Jensen不等式）：

考察 \(H(X) - \log_2 M\)：

\[ H(X) - \log_2 M = \sum_x p(x) \log_2 \frac{1}{p(x)} - \sum_x p(x) \log_2 M = \sum_x p(x) \log_2 \frac{1}{M p(x)} \]

利用不等式 \(\log_2 y \leq y - 1\)（当 \(y > 0\)），令 \(y = \frac{1}{M p(x)}\)，得：

\[ \log_2 \frac{1}{M p(x)} \leq \frac{1}{M p(x)} - 1 \]

两边乘以 \(p(x)\) 并求和：

\[ \sum_x p(x) \log_2 \frac{1}{M p(x)} \leq \sum_x \left( \frac{1}{M} - p(x) \right) = 1 - 1 = 0 \]

故 \(H(X) \leq \log_2 M\)，等号成立当且仅当 \(M p(x) = 1 \Rightarrow p(x) = \frac{1}{M}\)

💡 教授点拨：
“为什么等概率最‘混乱’？因为没有任何倾向性，最难预测。就像两个势均力敌的球队比赛，结果最让人捉摸不透。”

3.3.3 3. 实例对比：Machine 1 vs Machine 2

机器 字母 A B C D
Machine 1（等概） \(p(x)\) 0.25 0.25 0.25 0.25
Machine 2（非等概） \(p(x)\) 0.50 0.125 0.125 0.25

计算熵： - \(H_1 = \log_2 4 = 2\) bits - \(H_2 = 0.5\log_2 2 + 0.125\log_2 8 + 0.125\log_2 8 + 0.25\log_2 4 = 0.5 + 0.375 + 0.375 + 0.5 = 1.75\) bits

✅ 结论：等概分布的不确定性更高，信息熵更大。

🧠 深入理解：
聪明的Bob会根据概率设计提问顺序： - 先问高概率事件（如“是A吗？”），大概率一次命中； - 小概率事件虽需更多问题，但因发生概率低，对“平均问题数”影响较小。

这正是熵的本质：加权平均的自信息量。

3.4 四、相关概念扩展

3.4.1 1. 自信息量（Shannon Information Content）

定义：某一具体结果 \(x\) 发生时所携带的信息量： \[ h(x) = \log_2 \frac{1}{p(x)} \]

  • 特点：概率越小，信息量越大
  • 曲线趋势：随 \(p(x)\) 增大而单调递减
  • 当 \(p(x)=1\) 时，\(h(x)=0\)；当 \(p(x)\to0\) 时，\(h(x)\to\infty\)

💬 趣闻类比：
“有人告诉你‘中国队赢了巴西队’，你会震惊；但若说‘中国队输了’，你只会说‘哦，正常’。前者信息量远大于后者。”

3.4.2 2. 联合熵 \(H(X,Y)\)

衡量两个随机变量 \((X,Y)\) 联合发生的不确定性：

\[ H(X,Y) = -\sum_{x,y} p(x,y) \log_2 p(x,y) \]

若 \(X\) 与 \(Y\) 独立，则： \[ H(X,Y) = H(X) + H(Y) \]

3.4.3 3. 条件熵 \(H(X|Y)\)

在已知 \(Y\) 的条件下，\(X\) 仍具有的不确定性：

\[ H(X|Y) = \sum_y p(y) H(X|Y=y) = -\sum_{x,y} p(x,y) \log_2 p(x|y) \]

  • 若 \(X,Y\) 独立，则 \(H(X|Y) = H(X)\)
  • 若 \(X=Y\)，则 \(H(X|Y) = 0\)

3.4.4 4. 链式法则（Chain Rule）

联合熵可分解为： \[ H(X,Y) = H(Y) + H(X|Y) \]

同理： \[ H(X,Y) = H(X) + H(Y|X) \]

🔍 推导简述：

由 \(p(x,y) = p(y)p(x|y)\)，代入联合熵定义：

\[ H(X,Y) = -\sum_{x,y} p(x,y) \log_2 [p(y)p(x|y)] = -\sum_{x,y} p(x,y) \log_2 p(y) - \sum_{x,y} p(x,y) \log_2 p(x|y) \]

第一项 = \(-\sum_y p(y) \log_2 p(y) = H(Y)\)
第二项 = \(H(X|Y)\)
得证。

3.5 五、互信息（Mutual Information）：变量间的“共享信息”

3.5.1 1. 定义

互信息 \(I(X;Y)\) 衡量两个随机变量之间的依赖程度，即一个变量包含另一个变量信息的多少。

\[ I(X;Y) = \sum_{x,y} p(x,y) \log_2 \frac{p(x,y)}{p(x)p(y)} \]

⚠️ 注意符号：使用分号 ;，而非逗号 ,。逗号表示联合，分号表示互信息。

3.5.2 2. 互信息的性质

3.5.2.1 (1) 非负性：\(I(X;Y) \geq 0\)

🔍 证明（关键工具：Jensen不等式）

设函数 \(f(u) = -\log u\)，它是凸函数（convex）。

令 \(u = \frac{p(x)p(y)}{p(x,y)}\)，则：

\[ I(X;Y) = \mathbb{E} \left[ \log_2 \frac{p(x,y)}{p(x)p(y)} \right] = \mathbb{E} \left[ -\log_2 \frac{p(x)p(y)}{p(x,y)} \right] \]

由 Jensen 不等式（\(\mathbb{E}[f(U)] \geq f(\mathbb{E}[U])\)）：

\[ \mathbb{E} \left[ -\log_2 U \right] \geq -\log_2 \mathbb{E}[U] \]

计算 \(\mathbb{E}[U]\)：

\[ \mathbb{E}[U] = \sum_{x,y} p(x,y) \cdot \frac{p(x)p(y)}{p(x,y)} = \sum_{x,y} p(x)p(y) = 1 \]

所以：

\[ I(X;Y) \geq -\log_2 1 = 0 \]

✅ 等号成立当且仅当 \(X\) 与 \(Y\) 相互独立。

💡 教授点拨：
“Jensen不等式是信息论的‘瑞士军刀’。无论你是做机器学习、优化还是理论分析，都会频繁用到它。”

3.5.3 3. 互信息与其他熵的关系

\[ \begin{aligned} I(X;Y) &= H(X) - H(X|Y) \\ &= H(Y) - H(Y|X) \\ &= H(X) + H(Y) - H(X,Y) \end{aligned} \]

✅ 解释：
给你 \(Y\) 后，\(X\) 的不确定性减少了多少？这个减少量就是 \(X\) 和 \(Y\) 的互信息。

3.5.4 4. Venn 图理解熵的关系

        +---------------------+
        |         X           |
        |   +-------------+   |
        |   |   H(X|Y)    |   |
        |   |       +-----+---+------+
        |   |       | I(X;Y)| Y     |
        |   +-------+       |       |
        |           | H(Y|X)|       |
        +-----------+-------+-------+
                  H(X,Y)
  • 整个区域：联合熵 \(H(X,Y)\)
  • 左侧独有部分：\(H(X|Y)\)
  • 右侧独有部分：\(H(Y|X)\)
  • 中间重叠部分：互信息 \(I(X;Y)\)

✅ 独立情况：两圆无交集，\(I(X;Y)=0\)

3.6 六、连续随机变量：微分熵（Differential Entropy）

3.6.1 1. 定义

对于连续随机变量 \(X\)，概率密度函数为 \(f(x)\)，其微分熵为：

\[ h(X) = -\int f(x) \log_2 f(x) dx \]

⚠️ 重要区别： - 离散熵 \(H(X) \geq 0\) - 微分熵 \(h(X)\) 可正、可负、甚至无穷大

3.6.2 2. 常见分布的微分熵

3.6.2.1 (1) 均匀分布 \(X \sim \text{Uniform}(a,b)\)

  • PDF: \(f(x) = \frac{1}{b-a},\ x \in [a,b]\)
  • 微分熵： \[ h(X) = \log_2 (b-a) \]

✅ 直观解释：区间越宽，不确定性越高，熵越大。
❗ 当 \(b-a < 1\) 时，\(h(X) < 0\) —— 这在离散熵中不可能，但在连续情况下合理。

3.6.2.2 (2) 高斯分布 \(X \sim \mathcal{N}(\mu, \sigma^2)\)

  • PDF: \(f(x) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)\)
  • 微分熵： \[ h(X) = \frac{1}{2} \log_2 (2\pi e \sigma^2) \quad \text{(bits)} \]

🔍 推导要点：

使用自然对数先计算（单位为 nats）：

\[ h(X) = \frac{1}{2} \ln(2\pi e \sigma^2) \]

再换底：

\[ \log_2 u = \frac{\ln u}{\ln 2} \Rightarrow h(X)_{\text{bits}} = \frac{1}{2} \log_2 (2\pi e \sigma^2) \]

3.6.3 3. 最大熵定理（给定方差下）

在所有具有相同方差 \(\sigma^2\) 的连续分布中，高斯分布的微分熵最大。

即： \[ h(X) \leq \frac{1}{2} \log_2 (2\pi e \sigma^2) \]

💡 教授点拨：
“高斯噪声是‘最坏’的噪声，因为它携带最多的不确定性。这也是香农信道容量公式中出现高斯分布的原因。”

3.7 七、信源编码初步：从信息压缩谈起

3.7.1 1. 信源编码的目标

将离散信源输出的符号序列转换为二进制比特流，满足两个核心要求：

  1. 唯一可译性（Uniquely Decodable）
    接收端必须能无歧义地解码出原始符号序列。

  2. 高效性（Efficiency）
    最小化平均码长，提升传输效率。

✅ 教授强调：
“编码不是为了改变信息内容，而是为了更经济地表达它。”

3.7.2 2. 编码类型

类型 输入 输出 示例
定长码 固定长度符号序列 固定长度比特序列 ASCII码
变长码 单个符号 可变长度比特序列 摩尔斯电码
定长到变长 固定长度块 可变长度码字 Huffman编码
变长到变长 可变长度组 可变长度码字 Lempel-Ziv算法

3.7.3 3. 摩尔斯电码的智慧

观察国际摩尔斯电码： - 高频字母（如 E ·, T -）编码短 - 低频字母（如 Q --·-, Z --··）编码长

💡 设计哲学：
高频符号用短码，低频符号用长码，以降低平均码长。
这正是现代压缩编码（如Huffman）的核心思想。

3.7.4 4. 信源模型假设

本课程暂考虑最简单的信源模型： - 离散无记忆信源（Discrete Memoryless Source, DMS） - 时间离散 - 幅度离散 - 各符号独立同分布（i.i.d.）

3.8 八、总结与展望

3.8.1 本课核心知识图谱

概念 数学表达 物理意义
信息熵 \(H(X)\) \(-\sum p \log p\) 随机变量的不确定性
联合熵 \(H(X,Y)\) \(-\sum p(x,y) \log p(x,y)\) 两变量联合不确定性
条件熵 \(H(X\|Y)\) \(-\sum p(x,y) \log p(x\|y)\) 已知Y后X的剩余不确定性
互信息 \(I(X;Y)\) \(\sum p(x,y) \log \frac{p(x,y)}{p(x)p(y)}\) 两变量共享信息量
微分熵 \(h(X)\) \(-\int f \log f\ dx\) 连续变量的“不确定性”（可负）

3.8.2 关键方法论回顾

  1. 从直觉到数学：用“提问次数”建立对信息量的直观感知。
  2. 期望视角：熵是对“自信息量”的期望。
  3. 不等式工具：Jensen不等式是证明非负性的利器。
  4. 最大熵原理：约束条件下的最优分布。
  5. 编码思想：频率驱动压缩，追求平均性能最优。
